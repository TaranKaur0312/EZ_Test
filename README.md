## 🛠️ Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/TaranKaur0312/EZ_Test.git
cd EZ_Test
npm install
```

## 🚀 Running the App

Start the development server:

```bash
npm start
```

## 🏗️ Building for Production

To create a production build:

```bash
npm run build
```

## 🧪 Running Tests

Run tests using:

```bash
npm test
```
